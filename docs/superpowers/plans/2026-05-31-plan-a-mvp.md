# Plan A MVP Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build the first reliable Repo Context CLI release path: a TypeScript CLI that scans a local repository and generates conservative AI-agent context files.

**Architecture:** The CLI calls one orchestration function that scans repository facts, detects project signals, renders Markdown and JSON, and writes outputs through a single writer. Scanner and detector do not write files; renderers do not read from disk; writer owns dry-run and overwrite safety.

**Tech Stack:** Node.js, TypeScript, Vitest, GitHub Actions, npm package `bin`.
